introduction to js

javascript is an interpreted language hence we need an interpreter to execute js code

the browser comes in with bult in js engine (has an interpreter ) that translates javascript to machine code, feeds ther processor and gets back the feedback/result of the execution.

we can also download other javascript runtimes/translators /interpretors like node js, deno js or bun js to execute javascript outside the browser

js on the browser

to use js on the browser, create a html file with a script element

the script element can contain javascript code in it or src (source) attribute with the url path to a javascript(.js) file
